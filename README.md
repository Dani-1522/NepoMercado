# Nepomercado

Este repositorio contiene el backend y la aplicación móvil de Nepomercado.

## Estructura del repositorio

- `Backend/`: API RESTful construida con Node.js para la gestión de productos, usuarios y autenticación.
- `nepomercado_app/`: Aplicación móvil desarrollada en Flutter para la experiencia de usuario final.

## Instrucciones rápidas

### Backend

1. Ve a la carpeta `Backend`:
   ```bash
   cd Backend
   ```
2. Instala dependencias:
   ```bash
   npm install
   ```
3. Configura el archivo `.env` con tus variables de entorno.
4. Inicia el servidor:
   ```bash
   npm start
   ```

### App móvil (Flutter)

1. Ve a la carpeta `nepomercado_app`:
   ```bash
   cd nepomercado_app
   ```
2. Instala dependencias:
   ```bash
   flutter pub get
   ```
3. Ejecuta la app:
   ```bash
   flutter run
   ```

## Notas
- Asegúrate de tener Node.js, npm y Flutter instalados.
- Configura correctamente los archivos `.env` en ambos proyectos si es necesario.
- Consulta los README individuales en cada carpeta para más detalles.
